# create-khizab

Get up and running quickly with [Khizab](https://khizab.sh) by using the `create-khizab` CLI.

## Installation

```bash
npm create khizab
# or
pnpm create khizab
# or
yarn create khizab
```

## Documentation

For documentation and guides, visit [khizab.sh](https://khizab.sh/cli/create-khizab).
